The Parent Function Library

Every function family starts from a parent function — the simplest member of the family. Know these cold: their shape, key points, domain, and range.

Linear

$f(x) = x$
Domain: $(-\infty, \infty)$
Range: $(-\infty, \infty)$
Key: passes through origin, slope 1

Quadratic

$f(x) = x^2$
Domain: $(-\infty, \infty)$
Range: $[0, \infty)$
Key: vertex at origin, symmetric

Cubic

$f(x) = x^3$
Domain: $(-\infty, \infty)$
Range: $(-\infty, \infty)$
Key: inflection at origin, odd symmetry

Absolute Value

$f(x) = |x|$
Domain: $(-\infty, \infty)$
Range: $[0, \infty)$
Key: V-shape, vertex at origin

Square Root

$f(x) = \sqrt{x}$
Domain: $[0, \infty)$
Range: $[0, \infty)$
Key: starts at origin, increasing

Cube Root

$f(x) = \sqrt[3]{x}$
Domain: $(-\infty, \infty)$
Range: $(-\infty, \infty)$
Key: through origin, odd symmetry

Reciprocal

$f(x) = \dfrac{1}{x}$
Domain: $x \neq 0$
Range: $y \neq 0$
Key: hyperbola, asymptotes at axes

Exponential

$f(x) = 2^x$
Domain: $(-\infty, \infty)$
Range: $(0, \infty)$
Key: HA at $y=0$, through $(0,1)$

Logarithmic

$f(x) = \log_2(x)$
Domain: $(0, \infty)$
Range: $(-\infty, \infty)$
Key: VA at $x=0$, through $(1,0)$

Sine

$f(x) = \sin(x)$
Domain: $(-\infty, \infty)$
Range: $[-1, 1]$
Key: period $2\pi$, through origin

Cosine

$f(x) = \cos(x)$
Domain: $(-\infty, \infty)$
Range: $[-1, 1]$
Key: period $2\pi$, through $(0,1)$

Tangent

$f(x) = \tan(x)$
Domain: $x \neq \tfrac{\pi}{2} + n\pi$
Range: $(-\infty, \infty)$
Key: period $\pi$, VAs at odd $\tfrac{\pi}{2}$, through origin

The General Transformation Form

$$g(x) = a \cdot f\!\left(b(x - h)\right) + k$$
ParameterWhat It DoesExample
$a > 1$ Vertical stretch (taller) by factor $|a|$ $2x^2$ is twice as tall as $x^2$
$0 < |a| < 1$ Vertical compression (shorter) by factor $|a|$ $\tfrac{1}{2}x^2$ is half as tall
$a < 0$ Reflection over the $x$-axis (flips vertically) $-x^2$ opens downward
$h > 0$ Shift right by $h$ units $(x-3)^2$ shifts right 3
$h < 0$ Shift left by $|h|$ units $(x+2)^2$ shifts left 2
$k > 0$ Shift up by $k$ units $x^2 + 4$ shifts up 4
$k < 0$ Shift down by $|k|$ units $x^2 - 1$ shifts down 1
$b > 1$ Horizontal compression (narrower) by factor $\tfrac{1}{b}$ $(2x)^2$ is half as wide
$0 < b < 1$ Horizontal stretch (wider) by factor $\tfrac{1}{b}$ $(\tfrac{1}{2}x)^2$ is twice as wide
$b < 0$ Reflection over the $y$-axis (flips horizontally) $\sqrt{-x}$ is reflected left-right

Order of Transformations

When graphing $g(x) = a \cdot f(b(x-h)) + k$, read the transformations inside-out — just like PEMDAS tells you to start with the innermost parentheses:

$$\underbrace{a}_{\text{4th: multiply}} \cdot \underbrace{f\Big(\underbrace{b\big(\underbrace{x - h}_{\text{1st: shift}}\big)}_{\text{2nd: stretch}}\Big)}_{\text{3rd: apply }f} + \underbrace{k}_{\text{5th: shift}}$$
Think of it like PEMDAS: Just as $3(x-2)^2 + 5$ means "subtract 2, then square, then multiply by 3, then add 5," the transformation $a \cdot f(b(x-h))+k$ means: start closest to $x$ and work outward.

Follow the $x$: $$x \;\xrightarrow{\;\text{subtract }h\;}\; (x - h) \;\xrightarrow{\;\text{multiply by }b\;}\; b(x-h) \;\xrightarrow{\;\text{apply }f\;}\; f\!\big(b(x-h)\big) \;\xrightarrow{\;\text{multiply by }a\;}\; a \cdot f\!\big(b(x-h)\big) \;\xrightarrow{\;\text{add }k\;}\; \boxed{a \cdot f\!\big(b(x-h)\big) + k}$$
Why does this matter? If you do $b$ before $h$, you can accidentally shift to the wrong spot. Example: for $g(x) = (2x - 6)^2$, first rewrite as $(2(x-3))^2$ so you can see it's $h = 3$ then $b = 2$ — shift right 3, then compress horizontally by $\tfrac{1}{2}$. If you stretch first without rewriting, you'll misplace the vertex.

Example 3A: Graph $g(x) = -2(x - 3)^2 + 4$ by applying transformations to $f(x) = x^2$ in the correct order.

Given: $a = -2$, $b = 1$, $h = 3$, $k = 4$

Step 1 — Horizontal shift ($h = 3$): Closest to $x$ in the expression. Shift the entire parabola right 3 units. The vertex moves from $(0,0)$ to $(3,0)$. We now have $(x-3)^2$.

Step 2 — Horizontal stretch ($b = 1$): $b = 1$ means no horizontal scaling — skip this one.

Step 3 — Vertical stretch & reflect ($a = -2$): Stretch vertically by factor 2, then flip over the $x$-axis. The vertex stays at $(3,0)$ but the parabola now opens downward and is narrower. We have $-2(x-3)^2$.

Step 4 — Vertical shift ($k = 4$): Last operation, outermost. Shift everything up 4 units. The vertex moves from $(3,0)$ to $(3,4)$. Final: $g(x) = -2(x-3)^2 + 4$.

Showing: $f(x) = x^2$ (parent)

Example 3B: Graph $g(x) = \sqrt{-(x + 2)} + 1$ by applying transformations to $f(x) = \sqrt{x}$.

Given: Rewrite as $g(x) = \sqrt{-1 \cdot (x - (-2))} + 1$, so $a = 1$, $b = -1$, $h = -2$, $k = 1$.

Step 1 — Horizontal shift ($h = -2$): Innermost operation on $x$. Shift left 2 units. The starting point moves from $(0,0)$ to $(-2, 0)$. We have $\sqrt{x+2}$.

Step 2 — Horizontal reflect ($b = -1$): Reflect over the $y$-axis (through the shifted starting point). The square root now extends to the left from $(-2, 0)$. We have $\sqrt{-(x+2)}$.

Step 3 — Vertical stretch ($a = 1$): No change.

Step 4 — Vertical shift ($k = 1$): Outermost operation. Shift up 1 unit. Starting point moves to $(-2, 1)$. Final: $g(x) = \sqrt{-(x+2)} + 1$.

Showing: $f(x) = \sqrt{x}$ (parent)

Quick-Reference: How Key Points Move

If $(x_0, y_0)$ is a point on the parent $f(x)$, then the corresponding point on $g(x) = a \cdot f(b(x-h))+k$ is:

$$\left(\frac{x_0}{b} + h,\; a \cdot y_0 + k\right)$$

Where does this come from? Two parallel questions:

For $X$: "What input $X$ makes the inside of $f$ equal $x_0$?"   Solve $b(X - h) = x_0$:

$$X = \frac{x_0}{b} + h \quad \longleftarrow \text{only } b \text{ and } h \text{ affect the } x\text{-coordinate}$$

For $Y$: "Once $f$ outputs $y_0$, what does the outside of $f$ do to it?"   Apply $Y = a \cdot f(\cdots) + k$:

$$Y = a \cdot y_0 + k \quad \longleftarrow \text{only } a \text{ and } k \text{ affect the } y\text{-coordinate}$$
Clean symmetry: The horizontal parameters ($b$ and $h$) control where the point lands left–right. The vertical parameters ($a$ and $k$) control where it lands up–down. They don't cross over — $a$ and $k$ never touch the $x$-coordinate, and $b$ and $h$ never touch the $y$-coordinate.

This is the fastest way to plot a transformed function: take 3–5 key points from the parent, apply the formula, and connect the dots.

Problem 1: Describe the Transformations

Starting from $f(x) = x^2$, describe each transformation (in order) needed to obtain:

$$g(x) = -3(x + 2)^2 + 5$$

Rewrite: $g(x) = -3(x - (-2))^2 + 5$, so $a = -3$, $h = -2$, $k = 5$.

1. Shift left 2 units ($h = -2$)

2. Vertical stretch by factor 3 ($|a| = 3$)

3. Reflect over the $x$-axis ($a < 0$)

4. Shift up 5 units ($k = 5$)

Vertex: $(-2, 5)$. Parabola opens downward.

Problem 6: Graphing Exponential Transformations

Sketch the graph of the following function, labeling the asymptote, $y$-intercept, and one additional point:

$$g(x) = -2^{(x-1)} + 4$$

Parent: $f(x) = 2^x$. Here $a = -1$, $h = 1$, $k = 4$.

Asymptote: $y = 4$ (shifted up from $y = 0$).

Key point: Parent has $(0, 1)$. Transformed: $(0+1,\; -1 \cdot 1 + 4) = (1, 3)$.

Graph is decreasing (reflected), approaching $y = 4$ from below as $x \to -\infty$.

Sketch the graph of:

Problem 8: Cosine Transformation

Write the equation of a cosine function with amplitude 2, period $\pi$, phase shift right $\tfrac{\pi}{4}$, and midline $y = -3$.

Amplitude: $a = 2$.

Period: $\dfrac{2\pi}{b} = \pi \Rightarrow b = 2$.

Phase shift: $h = \dfrac{\pi}{4}$.

Midline: $k = -3$.

$$g(x) = 2\cos\!\left(2\left(x - \tfrac{\pi}{4}\right)\right) - 3$$

Simplified: $g(x) = 2\cos(2x - \tfrac{\pi}{2}) - 3$.
