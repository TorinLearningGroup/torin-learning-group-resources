# Econometrics

This section contains conceptual, graphical, and applied econometrics learning materials developed by Torin Learning Group.

Resources are designed to help students build strong statistical intuition for economic data, understand regression modeling deeply, and connect mathematical formulas with real empirical interpretation.

Topics may include:

- simple and multiple linear regression intuition  
- omitted variable bias and causal reasoning  
- hypothesis testing and confidence intervals  
- model specification and interpretation of coefficients  
- graphical and geometric views of least squares  
- ridge regression and regularization intuition  
- model comparison and predictive performance evaluation  
- data compression concepts and dimensionality reduction intuition  
- introductory machine learning ideas for economic modeling  
- probability foundations for econometric thinking
