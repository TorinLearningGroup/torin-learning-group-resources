# Statistics Resources

This section contains statistics and data analysis learning materials developed by Torin Learning Group.

Topics may include:

- descriptive statistics and data visualization  
- probability foundations and distributions  
- sampling methods and statistical inference  
- hypothesis testing and confidence intervals  
- correlation and regression analysis  
- conceptual intuition connecting formulas to real data reasoning  
- structured problem-solving approaches for exam preparation  

These resources are designed to help learners build deep conceptual understanding, mathematical confidence, and strong analytical thinking skills in statistics.
